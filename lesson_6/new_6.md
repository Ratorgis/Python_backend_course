# База Данных 

**База данных** — это упорядоченный набор структурированной информации или данных, которые обычно хранятся в электронном виде в компьютерной системе. База данных обычно управляется системой управления базами данных (СУБД). Данные вместе с СУБД, а также приложения, которые с ними связаны, называются системой баз данных, или, для краткости, просто базой данных.


# Про SQL

## Создаие таблицы на SQLite
Простое создание таблицы "book" с несколькоми полями

- `id` - будет автоматически подставлятся, с увеличением
- `title`, `autor`, `price`, `amount` - no null велечины, т.е. невозможно оставить при записи их пустыми 
- `awards` - можно осатвить пустой при записи, тогда будет просто `Null`


```SQL
CREATE TABLE book(
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    titile TEXT NOT NULL,
    autor TEXT NOT NULL,
    price INT NOT NULL,
    amount INT NOT NULL,
    awards TEXT
)
```

### Удаление

- полностью удалит таблицу
```sql
DROP TABLE book;
```

- удалит только ее содержимое
```sql
DELETE FROM book;
```

## Выбор или вывод из таблицы

- `SELECT` - можно выбрать все (*) или только что-то определенное 

```sql
SELECT id, titles FROM book;
```

## Запись в таблицу

## Выбор WHERE

Можно выбирать каждый раз над каким кортежем взаимодействовать, через филльтр `WHERE`
```sql
SELECT title FROM book WHERE id = 1 ;
```

## Все возможные операции в SQL

### Над строчками 

1. `LIKE` - для сравнению по подобию 
    - `%` - последующие символы любые
```sql
SELECT title FROM book WHERE title LIKE "Питон%"
```
